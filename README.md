# fishfeed
Software needed for automated fish feeding system of our fish facility

## How to install?

```bash
git clone https://github.com/royerlab/fishfeed.git
cd fishfeed
pip install -e .

# Check installation
fishfeed -h
```

## How to use?

```bash
# To check details of last 5 run
fishfeed last5

# To check details of last 50 run
fishfeed last50

# To run feeding
fishfeed run
```
